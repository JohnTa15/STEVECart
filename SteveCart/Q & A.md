- Πως θα λειτουργήσει το cart ?
> Θα εγκατασταθούν ultrasonic sensors, scales, monitor, ai cameras, nfc reader, λάμπα(σε περίπτωση blackout) και μικρόφωνο(για φωνητικές εντολές) και ηχείο
- Πως θα βοηθήσει τον πελάτη? 
> Θα μπορεί να βρίσκει το προιόν στους διαδρόμους που αναζητά ο πελάτης, θα μπορεί να βλέπει το συνολικό βάρος των προιόντων που έχει αγοράσει καθώς και την συνολική τιμή 
- Πως θα προσθέτει τα προιόντα και πως θα μπορεί να τα αφαιρεί ο πελάτης από το καλάθι? 
> Με την χρήση του nfc reader ο χρήστης θα μπορεί να σκανάρει και να περνάει το προιόν που επιθυμεί
- Τα προιόντα θα έχουν nfc tag πάνω τους ?
> Ναι και θα χρειαστεί ιδανικά να έχει υπολογιστεί από πριν τα βάρος τους(σε περίπτωση εξαπάτησης του καροτσιού ή γενικότερα ασφαλείας)
- Οι κάμερες ai που θα χρησιμεύσουν και που θα τοποθετηθούν? 
> Σε περίπτωση ασφάλειας κυρίως για τον καταστηματάρχη, αλλά και για τον ίδιο τον πελάτη. Θα υπάρχει μία κάμερα πίσω, μία που θα εστιάζει στο εσωτερικό του καροτσιού και άλλη μία που θα τοποθετηθεί στο μπροστινό μέρος του καροτσιού
- Το μικρόφωνο πότε θα ανοίγει και γιατί ?
> Το μικρόφωνο θα υπάρχει για διευκόλυνση του χρήστη. Να μπορεί να επικοινωνεί με έναν ψηφιακό assistant χωρίς να χρειάζεται να γράφει. Για παράδειγμα, ο πελάτης επιθυμεί καρότα θα ρώταει τον assistant σε ποιον διάδρομο και ράφι είναι τα καρότα για να μπορεί να τα βρει πιο εύκολα
- Γιατί να υπάρχουν ultrasonic sensors? 
> Για πιθάνα εμπόδια που μπορεί να βρει μπροστά το καρότσι για να μπορεί να ειδοποιεί τον πελάτη
- Γιατί να υπάρχουν λάμπες και ηχείο εγκατεστημένα στο καρότσι?
> Οι λάμπες για πιθανά blackouts μέσα στο supermarket. Το ηχείο θα μπορεί να θα υπάρχει για να ενημερώνει τους πελάτες και να τους προειδοποιεί(σε περίπτωση που μπορεί το σουπέρμαρκετ να κλείνει σε μερικά λεπτά ή σε περίπτωση πιθανών εμποδίων)
- Το σουπερμάρκετ θα έχει την δική του βάση με τα προιόντα τα οποία θα είναι διαθέσιμα ? επίσης θα περιέχει η βάση σε ποιο διάδρομο και ράφι θα βρίσκονται? 
> Ναι για θα χρησιμοποιηθεί mariadb για βάση δεδομένων που θα αναγράφει που θα βρίσκονται τα προιόντα, την ποσότητα τους, κλπ.
